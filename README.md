## Introduction
This repository contains the materials and analysis code for a systematic review and meta-analysis investigating the effects of transcranial alternating current stimulation (tACS). The project synthesizes results from randomized controlled studies comparing stimulation and sham conditions on cognitive outcomes. In particular, this project focuses on global cognition (MoCA and ADAS-Cog) and verbal memory (RAVLT-IR and RAVLT-DR). 

The repository includes:
- Description of the process and what was done 
- Extracted outcome data from the included studies
- R scripts used to calculate effect sizes and perform meta-analyses

All analyses were conducted in R using packages for meta-analysis and data processing. The goal of sharing this repository is to improve the transparency and reproducibility of the research by making the data processing and analytical workflow openly available.

*Note: This repository is still a work in progress and updates may be continuously made until the project has been completed.*
