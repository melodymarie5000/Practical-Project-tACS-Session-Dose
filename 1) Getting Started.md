# Getting Started

## Background and Review Rationale
Transcranial alternating current stimulation, or tACS, is an increasingly used non-invasive neuromodulation technique for treating a range of neurological and psychiatric disorders (Antal et al., 2013; Elymany et al., 2021; Pozdniakov et al., 2021).
However, when we look across the clinical trial literature, the results are quite mixed. Some studies report positive effects, others find no effect, and some report inconsistent or partial improvements.

One major reason for this variability is the substantial difference in how tACS is delivered across studies. Session dose parameters vary widely, including the number of sessions participants receive, how long each session lasts, the current amplitude used, and the length of the intersession interval.
Because these dosing parameters differ so much from one study to another, it becomes difficult to determine whether inconsistent clinical outcomes are due to the technique itself or simply due to differences in dose.

Substantial variability in session dose parameters
- Number of sessions 
- Session duration 
- Current amplitude
- Intersession interval

## Review Objective
This systematic review and meta-analysis aims to quantify the dose-response relationship between session dose parameters of tACS and clinical efficacy in clinical populations. 

*For this practical project, only an analysis on cognitive outcomes (MoCA, ADAS-Cog, and RAVLT) will be conducted and thus the information on this repository only contains the data and findings from studies with these outcomes and not the full dataset.*
