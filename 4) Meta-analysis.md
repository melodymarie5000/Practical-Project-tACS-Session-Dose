# Meta-analysis

The final step of this project involved conducting a meta-analysis of the included studies to quantitatively synthesize the results. The analysis was performed in R using the meta package, which provides functions for pooling effect sizes, assessing heterogeneity, and creating forest and funnel plots. The meta-analysis focused primarily on change scores between baseline and post-intervention, reflecting the effects of tACS across studies. The complete R script and analysis workflow are available in the repository here
, enabling full reproducibility of the results.
