# Meta-analysis

The final step of this project involved conducting a meta-analysis of the included studies to quantitatively synthesize the results. The analysis was performed in R using the meta package, which provides functions for pooling effect sizes, assessing heterogeneity, and generating forest and funnel plots. The meta-analysis primarily focused on change scores between baseline and post-intervention, reflecting the effects of tACS across studies. For outcomes measured on the same scale (RAVLT-IR and RAVLT-DR), the mean difference (MD) was calculated, while for global cognition outcomes (MoCA + ADAS-Cog), the standardized mean difference (SMD) was used to account for differences in measurement scales. The complete R script and analysis workflow are available in the repository [here](RScipt_Meta-Analysis.R).
